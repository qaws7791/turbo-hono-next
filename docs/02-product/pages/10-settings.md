# Settings (설정 다이얼로그)

## 개요

Settings는 사용자의 개인 설정, 계정 관리, 학습 환경을 설정하는 **모달 다이얼로그**입니다. 어느 페이지에서나 접근할 수 있도록 GNB 또는 아바타 메뉴를 통해 호출됩니다.

---

## 주요 설정 항목

### 1. 계정 (Account)

- 이메일 (읽기 전용)
- 표시 이름 변경
- 아바타 이미지 변경

### 2. 학습 환경 (Learning Environment)

- 선호하는 학습 시간 알림 (향후 확장)
- 기본 테마 설정 (System, Light, Dark)

### 3. 데이터 관리

- 로그아웃
- 계정 탈퇴 (좀비 데이터 정책에 따른 물리적 삭제 예약)

---

## UI/UX 원칙

### 즉시성

- 별도의 페이지 이동 없이 현재 화면 위에서 다이얼로그로 오버레이됩니다.
- 설정 변경 시 즉시 반영되거나, '저장' 버튼을 통해 명시적으로 업데이트합니다.

### 단순함

- 복잡한 탭 구조 대신 단일 뷰 또는 단순한 카테고리 분류를 사용합니다.

사용자가 학습 환경을 조정하는 곳입니다. 별도 페이지가 아닌 **Dialog(모달)** 형태로 제공하여 '설정 과잉'을 방지합니다.

---

## 의사결정 근거

- 사용자가 학습 환경을 조정하는 곳이지만, '설정 과잉'을 방지해야 함
- **"현재 화면을 유지한 채"** 설정을 변경하고 즉시 복귀하는 경험 제공
- 모달/다이얼로그 오버레이로 구현하여 흐름 끊김 방지

---

## 설계 목표

### 즉시 접근 + 컨텍스트 유지

- 어느 화면에서든 설정에 접근 가능
- 설정 후 원래 화면으로 즉시 복귀

### 학습 운영 중심

- 단순한 앱 설정이 아닌, '나만의 학습 OS 튜닝'
- 세션 길이, AI 스타일 등 학습에 특화된 설정

---

## 진입점 (Entry Points)

### 1. 사이드바 하단 사용자 메뉴

- ChatGPT/Sana Labs 스타일의 Popover
- 프로필/학습 개인화/설정/도움말/로그아웃 메뉴 통합
- 사용자의 플랜(Free/Pro) 상태 노출

### 2. 단축키

- `Cmd/Ctrl + ,` (Power User용)

---

## UI 구조 (Dialog)

### 데스크톱

- 중앙 고정 Dialog
- `max-w-3xl`

### Layout

- 2-컬럼: 좌측 네비게이션 + 우측 콘텐츠

### ScrollArea

- 좌측 네비와 우측 콘텐츠 영역에 각각 독립적인 스크롤 적용
- `h-[500px]`

### Interaction

- 즉시 저장 (Auto-save)
- Toast 피드백으로 저장 확인

## 주요 섹션 구성

| 섹션            | 주요 항목                                               |
| --------------- | ------------------------------------------------------- |
| **프로필**      | 이름, 이메일                                            |
| **일반**        | 테마(라이트/다크/시스템), 강조색(10종), 모션 감소       |
| **알림**        | 데일리 리마인더(시간/요일), 큐 생성 알림, 방해금지(DND) |
| **학습 개인화** | 기본 세션 길이(15~40분), AI 스타일, 난이도 자동 적응    |
| **데이터**      | 문서 보관 정책, 데이터 다운로드/삭제                    |

## 섹션별 상세

### 프로필

- 이름 편집
- 이메일 (읽기 전용)

### 일반

- 테마: 라이트 / 다크 / 시스템
- 강조색: 5가지 프리셋
- 모션 감소: 접근성 옵션

### 알림

- 데일리 리마인더: 시간, 요일 선택
- 큐 생성 알림: 새 할 일 생성 시
- 방해금지 시간대 설정

### 학습 개인화 (핵심)

- 기본 세션 길이: 15분 / 25분 / 40분 / 커스텀
- AI 스타일: 코치 / 소크라테스 / 친근한 튜터
- 난이도 자동 적응: 켜기 / 끄기

### 데이터

- 문서 보관 정책
- 데이터 내보내기 (다운로드)
- 계정 삭제 (데이터 삭제)

---

## UI 원칙

### Auto-save

- 변경 시 자동 저장
- Toast로 "저장됨" 피드백

### 최소 선택

- 각 설정은 명확한 기본값
- 고급 옵션은 접힌 상태

### 즉시 반영

- 테마 등 시각적 변경은 즉시 반영
- 미리보기 제공
