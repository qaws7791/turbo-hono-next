# 홈 (/home)

## 개요

홈은 사용자가 학습 애플리케이션에 접속했을 때 가장 먼저 마주하는 화면으로, **"오늘의 학습을 확인하고 즉시 시작하는 출발점"**입니다. 복잡한 의사결정 없이 현재 집중해야 할 과업을 직관적으로 제시하는 것을 목표로 합니다.

---

## 의사결정 근거

- **Focus & Start**: 사용자가 "무엇을 해야 하지?"를 고민하는 시간을 최소화합니다.
- **Motivation**: 학습 현황(완료, 연속 학습일)을 시각화하여 동기를 부여합니다.
- **Continuity**: 최근 학습한 내용을 타임라인으로 보여주어 학습의 흐름을 이어가게 합니다.

---

## 주요 구성 요소

### 1. 헤더 & AI 코칭 (Header & Coaching)

- **Greeting**: 시간대별(아침/오후/저녁) 인사와 사용자 이름을 표시하여 개인화된 경험을 제공합니다.
- **AI Coaching Message**: 사용자의 현재 학습 상태나 진행도에 맞춘 격려 메시지(예: "오늘도 잘하고 있어요!", "목표 달성까지 얼마 남지 않았어요!")를 노출합니다.

### 2. 현황 대시보드 (Stats Dashboard)

학습 현황을 한눈에 파악할 수 있는 3개의 카드 위젯을 제공합니다.

| 카드          | 내용                                              | 비고                                        |
| ------------- | ------------------------------------------------- | ------------------------------------------- |
| **남은 세션** | 오늘 수행해야 할 남은 세션 수 + 총 예상 소요 시간 | 학습 분량 파악                              |
| **오늘 완료** | 오늘 완료한 세션 수                               | 성취감 부여                                 |
| **연속 학습** | 연속 학습일(Streak) + 불꽃 아이콘 + 진행 바       | 게이미피케이션 요소 (최대 10일 기준 게이지) |

### 3. 오늘의 세션 (Session Queue)

**집계 대상**: 활성화된 모든 학습 계획(Active Plans) 중 오늘 예정된 세션

**노출 방식**:

- **요약**: 섹션 헤더에 총 세션 수를 표시합니다.
- **목록**: 최대 3개의 카드를 그리드(Grid) 형태로 노출합니다.
- **전체 보기**: 세션이 더 있을 경우 `/today` 페이지로 이동하는 링크를 제공합니다.

**카드 구성**:

- **Session Title**: 세션의 제목 (예: useState 정복하기)
- **Meta Info**: 포함된 계획명(Plan Title) 및 소요 시간(Duration)
- **Footer**: 계획 아이콘/색상 및 예정일(Scheduled Date)
- **Action**: 카드 클릭 시 해당 세션 플레이어 화면으로 즉시 이동

**Empty State (할 일이 없을 때)**:

- "오늘의 세션이 없습니다" 메시지 표시
- 학습 계획 생성 페이지(`/plans`)로 유도하는 CTA 버튼 제공

### 4. 최근 학습 (Recent Activity)

**목적**: 사용자가 이전에 학습한 내용을 상기하고 이력을 확인합니다.

**노출 방식**:

- **Timeline UI**: 수직 타임라인(Vertical Timeline) 형태
- **항목**: 최근 완료한 세션 (최대 6개)
- **내용**: 완료 일시, 모듈명, 세션명
- **Empty State**: 최근 기록이 없을 경우 안내 메시지 표시

---

## UI/UX 원칙

### 직관적인 탐색 (Intuitive Navigation)

- 가장 중요한 "오늘의 할 일"을 상단 중심에 배치하여 진입 장벽을 낮춥니다.
- 복잡한 필터나 설정을 제거하고, 시스템이 제안하는 최적의 순서를 따르도록 유도합니다.

### 시각적 피드백 (Visual Feedback)

- **Streak**: 불꽃 아이콘과 게이지를 통해 연속 학습을 시각적으로 보상합니다.
- **Card Color**: 각 학습 계획(Plan)의 고유 색상을 아이콘에 적용하여 시각적으로 구분합니다.

---

## 페이지 구조 (Layout)

```mermaid
graph TD
    Header[Greeting + AI Coaching]
    Stats[Stats Grid (Remaining / Completed / Streak)]
    Queue[Today's Sessions Grid (Max 3)]
    Timeline[Recent Activity Timeline]

    Header --> Stats
    Stats --> Queue
    Queue --> Timeline
```
