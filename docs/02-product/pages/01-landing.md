# 랜딩 페이지 (/)

## 개요

랜딩 페이지는 **첫 방문자를 유료 사용자로 전환**하기 위한 핵심 접점입니다.

> **핵심 메시지**: "자료만 업로드하세요. 나머지는 AI가 합니다."

---

## 의사결정 근거

- 신규 사용자가 제품의 가치를 **3초 내에** 이해할 수 있어야 함
- 2025년 랜딩 페이지 트렌드 적용: AI 개인화, 몰입형 인터랙션, 모바일 퍼스트, 미니멀리즘
- "Calm Focus OS" 디자인 컨셉과 일관성 유지하면서 전환율 극대화

---

## 구성 섹션 (10개)

### 1. Navigation

- Sticky 헤더
- Glassmorphism 효과 (Backdrop blur)
- 로고, 요금제 링크, "무료로 시작하기" 버튼

### 2. Hero (Social Proof 포함)

- **핵심 가치 제안**: "자료만 업로드하세요. 나머지는 AI가 합니다."
- **서브 카피**: "목표를 말하고 자료를 올리면, AI가 경로를 설계하고 매일 세션을 준비합니다."
- **Primary CTA**: "무료로 시작하기" / "사용법 보기"
- **Social Proof**: 평점 4.9/5.0, 사용자 1,200+, 학습 시간 절감 35%
- **앱 프리뷰**: 카드 형태의 프리뷰 영역

### 3. Pain Points (사용자의 고통)

- **타이틀**: "사용자의 고통을 줄입니다"
- **4가지 문제점**:
  1. **무엇부터?**: 자료는 많은데, 경로가 없음
  2. **꾸준함**: 매번 계획을 세우다 지침
  3. **진행 추적**: 학습 진도 파악의 어려움
  4. **정리**: 노트 노동의 부담

### 4. Solutions (해결책)

- **타이틀**: "해결책은 4가지"
- **4가지 핵심 기능**:
  1. **자동 경로 설계**: 자료 분석 후 모듈/세션 단위 경로 구성
  2. **적응형 세션**: 이해도 확인 및 맞춤 힌트 제공
  3. **진행 상황 대시보드**: 학습 진도와 달성률 시각화
  4. **지식 아카이브**: 개념 자동 정리 및 검색 중심 제공

### 5. How It Works (준비 과정)

- **타이틀**: "준비 과정"
- **4단계 사용법**:
  1. **자료 업로드**: PDF/URL/텍스트 업로드
  2. **분석 및 설계**: AI가 구조와 개념 추출
  3. **학습 계획 생성**: 목표/수준 선택 후 즉시 생성
  4. **매일 시작**: Home에서 '시작' 버튼 클릭

### 6. Science Backed (과학적 근거)

- **타이틀**: "과학적 근거"
- **3가지 원리**:
  - 인지 부하 이론
  - 인출 연습
  - 지식 추적

### 7. Pricing (요금제)

- **타이틀**: "요금제"
- **플랜 구성**:
  - **무료**: ₩0, 스페이스 2개, 문서 10개
  - **프로**: ₩19,000, 무제한 학습 계획, 고급 세션 개인화
  - **팀**: 문의, 팀 프로젝트, 관리/보안 옵션

### 8. FAQ

- **타이틀**: "FAQ"
- **4개 아코디언 항목**:
  1. 자료 업로드 시 학습 계획 자동 변경 여부
  2. 세션 상세 페이지 유무 (세션 요약 카드로 대체)
  3. 데이터 안전성 (로컬 mock 데이터 동작)
  4. 모바일 지원 여부

### 9. Final CTA

- **타이틀**: "지금 시작하세요"
- **메시지**: "오늘의 세션은 AI가 준비합니다"
- **버튼**: "무료로 시작하기"

### 10. Footer

- 로고 및 Copyright
- 링크: 소개, 이용약관, 개인정보처리방침
