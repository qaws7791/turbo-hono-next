# 오늘 할 일 (Today) (/today)

## 개요

오늘 할 일 페이지는 **오늘의 세션**을 확인하고 관리하는 화면입니다.
홈 화면에서 요약된 형태로 보이던 전역 큐를 **전체 목록으로 펼쳐서** 확인하며, 오늘 완료해야 할 모든 학습 세션을 한눈에 볼 수 있습니다.

---

## 의사결정 근거

- **집중과 확장**: 홈은 "시작"에 집중하여 핵심 정보만 노출하고, 상세 목록은 별도 페이지(/today)에서 제공합니다.
- **전역 뷰**: 특정 계획(Plan)에 얽매이지 않고, "오늘"이라는 시점 기준으로 내가 해야 할 모든 학습을 모아서 보여줍니다.
- **명확한 현황**: 전체 할 일의 개수와 완료된 개수를 상단에 배치하여 성취도와 남은 분량을 직관적으로 파악하게 합니다.

---

## 구성 요소

### 1. 헤더 (TodayHeader)

- **제목**: "오늘의 세션"
- **설명**: "오늘 완료해야 할 모든 학습 세션입니다."
- **네비게이션**: 상단 공통 헤더를 사용 (뒤로가기 등 포함)

### 2. 현황 통계 (TodayStats)

오늘 할 일 큐의 요약 정보를 표시합니다.

- **전체**: 오늘 예정된 총 세션 수
- **완료**: 오늘 이미 완료한 세션 수

### 3. 할 일 목록 (TodayQueueList)

각 항목은 **카드(Card)** 형태로 표시되며, 다음 정보를 포함합니다.

| 필드               | 설명                                   |
| ------------------ | -------------------------------------- |
| **세션 제목**      | 학습할 세션의 이름 (클릭 시 학습 시작) |
| **Plan 정보**      | 소속된 계획의 아이콘, 색상, 제목       |
| **예상 소요 시간** | 분 단위 (예: 25분)                     |
| **예정일**         | 오늘 날짜로 표시                       |

> **디자인 요소**: 각 카드는 소속된 Plan의 고유 색상(아이콘 색상)을 통해 시각적으로 구분이 가능합니다.

### 4. 빈 상태 (Empty State)

오늘 할 일이 없을 경우 표시되는 화면입니다.

- **메시지**: "오늘의 세션이 없습니다 🎉"
- **가이드**: "학습 자료를 업로드하고 학습 계획을 생성해보세요."

---

## 인터랙션

1. **학습 시작**
   - 세션 카드의 **제목**을 클릭하면 학습 세션 페이지(`session?sessionId=...`)로 이동합니다.
   - 이동 후 바로 학습(풀스크린 모드)을 시작할 수 있습니다.

2. **완료 상태 확인**
   - 세션을 완료하면 목록에서 상태가 업데이트되거나(현재 구현상 큐에서 제거되거나 완료 표시), 통계의 '완료' 카운트가 증가합니다.
