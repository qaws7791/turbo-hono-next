# Concept Library (지식 아카이브) (/concepts)

## 개요

Concept Library는 학습 과정에서 생성되는 **Concept(핵심 개념)**을 자동으로 저장하고, 검색과 복습의 기준점으로 사용하는 지식 아카이브입니다.

---

## 의사결정 근거

- 장기 기억과 재사용은 ‘세션’이 아니라 ‘개념(Concept)’ 단위로 일어남
- 사용자가 직접 정리하도록 만들면 학습이 아니라 노트 노동이 됨
- 자동 저장 + 검색 중심의 단순한 구성이 가장 빠르게 쌓이고 오래 감

---

## 기능

### 자동 저장

- 세션 완료 시 Concept가 자동 생성/업데이트
- Concept에는 다음이 포함될 수 있음:
  - 정의
  - 예제(코드/상황)
  - 주의사항(오해/실수 포인트)
  - 태그
  - 학습 출처(어떤 세션에서 나왔는지)

### 복습 관리

- Concept마다 복습 필요도를 표시
- “복습 시작”은 관련 세션으로 바로 진입할 수 있어야 함

### 검색

- 검색창 + 카드 리스트 중심
- 복잡한 폴더/수동 분류는 제공하지 않음

---

## Concept 카드 구성

| 필드       | 설명                                 |
| ---------- | ------------------------------------ |
| 제목       | 개념 이름                            |
| 핵심 1줄   | 한 줄 요약                           |
| 복습 상태  | 양호 / 곧 복습 / 복습 필요           |
| 태그       | 관련 키워드                          |
| 출처(선택) | 최신 학습 세션의 모듈/세션 정보 표시 |

---

## Space 관점

- 전역 Concept Library는 `/concepts`에서 전체 Space 기준으로 제공
- Space 상세의 “개념” 탭(`/spaces/:spaceId/concepts`)에서는 해당 Space의 Concept만 확인

---

## 복습 상태

| 상태      | 설명           |
| --------- | -------------- |
| 양호      | 충분히 기억 중 |
| 곧 복습   | 복습 시점 임박 |
| 복습 필요 | 지금 복습 권장 |

---

## 검색 기능

### 검색 대상

- Concept 제목
- 1줄 요약
- 태그

### 필터 옵션

- (현재 프로토타입) 검색어 필터만 제공
- 복습 상태 필터/정렬은 추후 확장
