# Plan 상세 (운영 콘솔) (/spaces/:spaceId/plan/:planId)

## 개요

Plan 상세는 **실행(오늘/다음 세션 시작) + 안내(경로/현재 위치)**를 최소 UI로 제공하는 "Plan 운영 콘솔"입니다.

---

## 의사결정 근거

- 세션 상세 페이지를 만들지 않는 대신, Plan 상세가 **실행 + 운영 + 안내**를 최소 UI로 제공
- 핵심 루프가 "Home→Session View"이고, 사용자는 매일 "시작"만 누르는 경험 기대
- Plan 상세는 "관리 화면"이 아니라 "시작 버튼으로 이어지는 안내/운영 콘솔"
- 기존 경로 안내를 Plan 상세 내부로 통합 (Curriculum으로 재구성)

---

## 구성 섹션

### 1. Header

**표시 항목**:

- Plan 제목
- 기간 타입
- 상태 (`active` / `paused` / `archived`)
- Active 강조 (배지)

**CTA 버튼**:
| 버튼 | 동작 |
|------|------|
| `세션 시작` (Primary) | 풀스크린 Session View 진입 |
| `Pause` / `Resume` | 상태 토글 |
| `Archive` | 보관 처리 |

### 2. Home / Next 실행 큐

오늘 해야 할 세션 1~3개 표시:

| 필드            | 설명                                        |
| --------------- | ------------------------------------------- |
| 예정일          | 해당 세션 날짜                              |
| 유형            | 세션 / 복습                                 |
| 예상 소요 시간  | 예: 25분                                    |
| 모듈명 + 세션명 | 예: "Module 2: Hooks / Session 1: useState" |

각 항목 행동은 1개: `시작` (세션은 풀스크린 View로)

### 3. Progress & Rhythm

- 전체 진행률 (%)
- 이번 주 흐름 (간단 바)
- 지연/복습 적체는 "상태 칩" 수준으로만 표시
- 과한 관리 UI 금지

### 4. Curriculum (Modules)

학습 계획의 모듈 구조를 표시하여 전체 학습 경로를 안내

**구성 요소**:

- 현재 위치 강조 (현재 모듈/현재 세션)
- 목표 요약 + 전체 진행률
- 모듈 리스트 (접이식/아코디언)
  - 모듈 제목, 요약, 진행률(세션 완료 수 기반)
  - 모듈 펼침 시 세션 리스트 표시
    - 세션 제목, 유형(learning/review), 예정일, 예상 시간, 상태
    - **인터랙션**: 완료된(Completed) 세션 클릭 시 → **세션 요약 뷰(Concept Library의 해당 세션 필터 화면)**로 이동하여 학습 내용 복습 가능
- `다음 세션 시작` CTA

**UI 기준**:

- 수동 편집 불가
- 트리/복잡한 목차 UI 금지, 단순 아코디언 중심
- 상세는 기본적으로 접고, 필요 시 최소 정보만 확장

## 상태별 UI

### Active Plan

- "세션 시작" 버튼 활성화
- Home 큐 표시
- 진행률 강조

### Paused Plan

- "Resume" 버튼 표시
- "일시정지 상태입니다" 메시지
- 진행률은 유지 표시

### Archived Plan

- "이 Plan은 보관되었습니다" 메시지
- 복원 옵션 제공
- 읽기 전용
