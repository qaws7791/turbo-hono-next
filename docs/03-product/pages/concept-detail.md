# Concept 상세 (/concept/:conceptId)

## 개요

Concept 상세는 아카이브에 저장된 개념을 **빠르게 읽고, 복습을 시작하고, 학습 이력을 확인**하는 화면입니다.

---

## 구성 요소

### 1) 헤더

- Concept 제목
- 1줄 요약
- 복습 상태 배지(양호/곧 복습/복습 필요)
- CTA: “복습 시작”
  - 가능할 때: 가장 최근 학습 출처(세션)로 바로 진입
  - 불가능할 때: 비활성(출처가 없을 때)

### 2) 태그

- 자동 생성된 태그를 칩 형태로 표시

### 3) 탭

```
[Ari 노트] [학습 이력] [연관 개념]
```

---

## Ari 노트 탭

**목적**: 학습한 내용을 “복습용 한 페이지”로 정리해 제공

- 정의(본문)
- 예제(코드/예시)
- 주의사항(실수/오해 포인트)

> Ari 노트는 사용자가 직접 편집하지 않습니다(읽기 전용).

---

## 학습 이력 탭

**목적**: 이 Concept를 “언제, 어떤 세션에서” 학습했는지 추적

- 학습 이력 목록(최신순)
  - 학습 일시
  - 세션 제목
  - 모듈 제목
- 항목 클릭 시: 해당 세션으로 이동해 다시 학습/복습 가능

---

## 연관 개념 탭

**목적**: 함께 보면 좋은 관련 Concept로 빠르게 확장

- 연관 Concept 목록
- 클릭 시 해당 Concept 상세로 이동

---

## UI 원칙

- 읽기 중심(편집 기능 최소화)
- “복습 시작”이 항상 상단에서 보이도록 배치
- 브레드크럼으로 현재 위치(전역 라이브러리/Space) 맥락 유지
