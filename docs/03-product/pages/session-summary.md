# 세션 요약 (Session Summary)

## 개요

별도의 “세션 상세 페이지”를 만들지 않고, 홈의 **최근 학습(타임라인)**과 Concept Library를 통해 “방금 배운 것 다시 보기”를 해결합니다.

---

## 의사결정 근거

- 세션 상세가 쌓이면 결국 LMS형 “레슨 목록”이 됨
- 사용자는 “방금 배운 것”만 빠르게 다시 확인하길 원함
- 지식은 Concept에 저장하고, 세션은 묶음/기록으로만 남기는 편이 단순함

---

## 노출 위치

### 1) 홈 (Home) > 최근 학습

- 최근 완료한 세션을 시간순으로 표시
- 항목 클릭 시: 해당 세션에서 만들어진/업데이트된 Concept로 이동

### 2) Plan 상세 > 커리큘럼

- 완료된 세션은 “요약 보기”로 이동 가능

---

## “요약 보기” 동작

1. Concept Library로 이동
2. (현재 프로토타입) 전역 Concept 목록에서 복습을 이어감
3. (추후 확장) 세션 기준으로 묶어 보는 UI로 더 빠르게 “방금 배운 것”만 확인

---

## UI 원칙

- 가벼운 요약(필수 정보만)
- 상세 내용은 Concept 중심으로 제공
- 최신 항목이 상단에 오도록 정렬
