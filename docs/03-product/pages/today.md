# 오늘 할 일 (Today) (/today)

## 개요

오늘 할 일 페이지는 홈에서 일부만 보이던 **전역 큐를 전체 목록으로 펼쳐서** 확인하는 화면입니다.

---

## 의사결정 근거

- 홈은 "시작"에 집중해야 하므로, 긴 목록을 모두 노출하지 않음
- 할 일이 많을 때는 별도 화면에서 전체를 확인하는 편이 명확함
- 전역 큐는 메뉴를 이동하지 않아도 모든 계획의 할 일을 한눈에 확인할 수 있어야 함

---

## 구성 요소

### 1) 헤더

- 제목: "오늘 할 일"
- 설명: "오늘 완료해야 할 모든 학습 세션 목록"
- 뒤로가기: "홈으로"

### 2) 간단 통계

- 전체 개수
- 세션 개수

### 3) 할 일 목록

각 항목은 카드 형태로 표시됩니다.

| 필드           | 설명                    |
| -------------- | ----------------------- |
| 유형           | 세션                    |
| 세션 제목      | 오늘 수행할 항목의 제목 |
| Plan 이름      | 어떤 계획의 항목인지    |
| 예상 소요 시간 | 분 단위(예: 25분)       |
| 예정일(날짜)   | 오늘 기준으로 정렬/표시 |

**액션**: 항목 클릭 → 바로 학습 세션 시작(풀스크린 세션 모드)

---

## 빈 상태

오늘 큐가 비어 있을 때:

```
오늘 할 일이 없습니다.

학습 자료를 업로드한 다음,
나만의 학습 계획을 생성해보세요.
```
