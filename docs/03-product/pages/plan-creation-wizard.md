# Plan 생성 위저드 (/plans/new)

## 개요

Plan 생성 위저드는 **3단계 위저드** 형태로, 사용자가 "무엇을 배울지/어떻게 배울지"를 고민하는 시간을 최소화하고 Plan 생성 완료까지의 경로를 **1분 내**로 줄입니다.

---

## 의사결정 근거

- 고민 시간 최소화 → 빠른 Plan 생성
- Plan은 선택한 문서들의 집합을 기반으로 생성 → 이후 다른 문서가 추가되어도 기존 Plan은 영향받지 않음
- **즉시 생성**: Preview 단계 없이 바로 Plan 생성하여 불필요한 확인 절차 제거
- **사용자 통제감 유지**: AI가 계획을 제안하지만 최종 결정은 사용자가 함

---

## 진입점 (Entry Points)

1. **학습 계획 (Plans) 목록 페이지**: "Plan 만들기" Primary CTA
2. **홈 (Home)**: Active Plan 없을 때 "Plan 만들기" CTA
3. **빠른 액션 카드**: "새 Plan 생성" 버튼

---

## 라우팅

| 경로             | 설명                                |
| ---------------- | ----------------------------------- |
| `GET /plans/new` | Plan 생성 위저드                    |
| 완료 후          | `/plans/:planId` (Plan 상세)로 이동 |

---

## 화면 형태

- **데스크톱**: 중앙 카드 (`max-w-4xl`)
- **모바일**: 1컬럼, 상단 Step 표시 (번호 + 텍스트)

---

## Step 1: 문서 선택

**목적**: Plan의 입력(학습 자료)을 결정

### UI 구성

- 상단: 검색 입력 ("문서 제목/태그/요약 검색")
- 문서 카드 리스트:
  - 제목 / 유형 아이콘
  - 상태 배지: `분석 완료`, `분석 중`, `미분석`
  - 1줄 요약
  - 체크박스 선택

### 제약/검증

- **최소 1개, 최대 5개** 선택 가능
- "미분석","분석 중" 상태의 문서 포함 시: 진행 불가

### 빈 상태

```
Plan을 만들려면 학습 자료가 필요합니다.

- "학습 자료" 메뉴로 이동하여 자료를 업로드한 뒤 다시 돌아옵니다.
```

---

## Step 2: 학습 목표 및 수준 설정

**목적**: 사용자의 학습 의도와 현재 수준 파악

### UI 구성

**학습 목표** (카드 선택 - 단일):

- `취업/이직`: 실무 중심, 면접 대비
- `자격증 취득`: 시험 범위 핵심, 기출 유형
- `업무 적용`: 즉시 사용 가능한 팁, 문제 해결
- `취미/교양`: 흥미 위주, 폭넓은 이해

**현재 수준** (슬라이더 또는 세그먼트 버튼):

- `완전 초보`: 비전공자/입문자 수준 (용어 설명 포함)
- `기초`: 기본 개념 이해 (응용 필요)
- `중급`: 실무 경험 있음 (심화/디테일 필요)
- `고급`: 전문가 수준 (트렌드/엣지 케이스)

---

## Step 3: 목표 기한 및 요구사항

**목적**: 현실적인 스케줄링과 커스터마이징

### UI 구성

**목표 기한** (라디오 버튼 + 입력):

- `원하는 기간 입력`: 일/주/월 단위 직접 입력 (예: 2주 안에 끝내고 싶어요)
- `학습량에 맞춤`: AI가 분량을 분석하여 적정 기간 제안

> 참고: 현재 프로토타입 UI에서는 “미리보기”가 별도로 제공되지 않습니다.

**특별 요구사항** (선택사항, 텍스트 영역):

- 예: "주말에는 공부 못해요", "실습 위주로 짜주세요", "파이썬 코드 예시 많이 넣어주세요"
- Placeholder 제공으로 입력 유도

### 생성 처리 (Action)

- **Primary Action**: "Plan 생성하기" (Sparkles 아이콘)
- **처리 흐름**: 버튼 클릭 시 로딩 상태(Skeleton/Spinner)로 전환되며 AI가 Plan 생성
- **완료 후**: 생성된 Plan 상세 페이지로 자동 이동

---

## 사용자 흐름 및 완료 처리

### 흐름 단계

1. **시작**: 위저드 진입
2. **문서 선택**: 학습할 자료 고르기
3. **설정**: 목표와 수준, 기한 정하기
4. **생성 중**: AI가 계획을 짜는 동안 로딩 표시
5. **완료**: 생성된 계획 상세 화면으로 이동
