# 플랫폼 비전

## 개요

**AI 개인화 학습 플랫폼**은 학습자가 "무엇을 배울지, 어떻게 배울지"에 대한 의사결정 부담을 제거하고, AI가 학습 전 과정을 자동으로 설계·운영하는 **개인 학습 운영체제(Learning OS)**입니다.

> "자료만 업로드하세요. 나머지는 AI가 합니다."

---

## 핵심 가치 제안

### 1. 의사결정의 AI 이동

사용자가 "무엇을 배울지/어떻게 배울지"를 결정하는 순간 학습 지속률이 떨어집니다. 이 플랫폼은 의사결정을 AI로 이동시켜 사용자의 인지 부담을 최소화합니다.

### 2. LMS 복잡성 제거

전통 LMS의 복잡한 설정, 콘텐츠 관리, 코스 구조 설계는 개인 학습자에게 과도한 부담입니다. 이를 제거하고 단순한 인터페이스만 제공합니다.

### 3. 근거 기반 신뢰

"경험하지 않아도 이해되는 신뢰"를 확보하기 위해 러닝사이언스 원리에 기반한 설계를 적용합니다.

---

## 핵심 원칙

| 원칙              | 설명                                            |
| ----------------- | ----------------------------------------------- |
| **최소 행동**     | 사용자는 자료만 업로드하고, 나머지는 AI가 담당  |
| **자동화된 설계** | 학습 과정의 의사결정과 설계는 AI가 수행         |
| **단순한 구조**   | 플랫폼 구조는 단순하며, 페이지 수는 최소로 유지 |

---

## 이론적 배경

이 플랫폼은 다음의 러닝사이언스 이론에 기반합니다:

- **Knowledge Space Theory**: 지식 공간 이론
- **Learning Progressions**: 학습 진행 이론
- **Cognitive Load Theory**: 인지 부하 이론
- **Retrieval Practice**: 인출 연습
- **Knowledge Tracing**: 지식 추적
- **PKM (Personal Knowledge Management)**: 개인 지식 관리 (Second Brain/Atomic Notes)

---

## 목표 사용자

이 플랫폼은 다음과 같은 사용자를 대상으로 합니다:

- **자기주도 학습자**: 독립적으로 학습하지만 계획 수립에 어려움을 겪는 사람
- **시간 제약이 있는 학습자**: 효율적인 학습을 원하는 바쁜 직장인
- **지식 축적을 원하는 학습자**: 학습 내용을 체계적으로 정리하고 싶은 사람

---

## 핵심 차별점

| 기존 LMS                | Learning OS                    |
| ----------------------- | ------------------------------ |
| 수십 개의 메뉴/페이지   | 핵심 화면 + 풀스크린 세션 모드 |
| 사용자가 코스 구조 설계 | AI가 학습 경로 자동 생성       |
| 수동 복습               | AI 기반 맞춤형 학습            |
| 콘텐츠 중심             | 학습 루프 중심                 |
| "시험" 느낌의 진단      | 세션 내 자연스러운 이해도 체크 |
