# 용어집

> 이 문서는 프로젝트에서 사용되는 주요 용어와 개념을 정의합니다.

---

## 플랫폼 용어

| 용어            | 설명                                                                                                                                                                         |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Learning OS** | 개인 학습 운영체제. AI가 학습 전 과정을 자동으로 설계·운영하는 플랫폼                                                                                                        |
| **Material**    | 학습 자료. 사용자가 업로드하는 PDF, 텍스트 등의 원본 데이터 (구 Documents)                                                                                                   |
| **Plan**        | 학습 계획 및 실행 단위. 특정 학습 의도나 목표를 달성하기 위해 사용자가 선택한 Material(자료)을 바탕으로 생성됨                                                               |
| **Module**      | 학습 모듈. Plan 내부의 의미 단위 묶음(Phase/Chapter). 학습 목표/범위/완료 조건을 가진 커리큘럼의 상위 블록<br/>_예: "Module 1: Fundamentals", "Module 2: Advanced Concepts"_ |
| **Session**     | 학습 세션. 20~40분 단위의 몰입형 학습 시간. 반드시 하나의 Module에 소속됨                                                                                                    |

---

## 주요 기술 개념

제품의 스마트한 기능을 가능하게 하는 핵심 기술들입니다.

| 용어                           | 쉬운 설명                                                                                                   |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| **문서 기반 답변 (RAG)**       | 사용자가 올린 문서를 AI가 미리 읽고, 그 내용을 바탕으로 정확하게 답변을 생성하는 방식입니다.                |
| **의미 분석 (Embedding)**      | 글자 그대로의 의미가 아닌, 문맥과 뜻을 AI가 이해할 수 있는 수학적 형태로 변환하는 과정입니다.               |
| **지식 저장소 (Vector Store)** | AI가 학습 자료의 뜻과 맥락을 이해하고, 필요할 때 빠르게 찾아낼 수 있도록 정리해둔 똑똑한 도서관과 같습니다. |
| **AI Model (Google Gen AI)**   | Google의 Gemini 모델을 사용하여 텍스트 생성, 임베딩 등 핵심 AI 기능을 수행합니다.                           |

## 러닝사이언스 용어

| 용어                       | 설명                                                 |
| -------------------------- | ---------------------------------------------------- |
| **Cognitive Load Theory**  | 인지 부하 이론. 작업기억의 한계를 고려한 학습 설계   |
| **Knowledge Space Theory** | 지식 공간 이론. 지식 간의 선행 관계를 모델링         |
| **Retrieval Practice**     | 인출 연습. 정보를 기억에서 끄집어내는 연습           |
| **Learning Progressions**  | 학습 진행 이론. 개념의 발전 경로를 정의              |
| **Knowledge Tracing**      | 지식 추적. 학습자의 현재 지식 상태를 모델링          |
| **PKM**                    | Personal Knowledge Management. 개인 지식 관리 시스템 |
